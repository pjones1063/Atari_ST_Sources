LDG bindings for GFABASIC by Lonny Pursell
Released: 4/15/2015
Updated: 4/18/2015

You need library COOKIES.LDG installed to run the example code.

LDG_LIB.GFA  -> Access LDG libraries via the LDG TSR/Cookie method.
  This code was derived from the LDG hyptext documentation regarding the
  cookie access.

LDG_EXEC.GFA -> Access LDG libraries without installing the LDG TSR/Cookie.
  This code was derived by porting the LDG lib used to link 'C' programs.
