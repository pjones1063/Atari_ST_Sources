The main documentation is in FLDLIB.DOC. This file details the 
alterations to FLDLIB between versions.

17/4/94   v0.9 beta
-------------------

First released version.


7/5/94    v0.91 beta
--------------------

Two calls added, fld_findtype() and fld_findform().

Bugfix for fld_key() and fld_mouse(): both bombed if key pressed/mouse 
clicked on window with no floating dialogs open.


10/5/84   v0.92 beta
--------------------

Bugfix for FLDC_MOUSE mode of fld_open(): opened dialogues can no 
longer overlap menu bar.
