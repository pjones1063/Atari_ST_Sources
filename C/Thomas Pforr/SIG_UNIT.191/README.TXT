
  SIG_UNIT ist ein Modul zur Ausgabe von SIGNUM-Editor-Fonts (*.E24)
  in eigenen C-Programmen.

  Der Orginalquellcode wurde in der ct 6.90 S. 290 ff ver”ffentlicht.




  SHOW_SIG.PRG  dient zur Anzeige von *.E24.
  Nach Aufruf des Programmes kann in der Fileselectbox ein
  Pfadname und eine Maske eingegeben werden.

  z.B. Pfad  = C:\\CHSETS\\*.E24
       Maske = *.E24                    -> zeigt alle Fonts in diesem
                                           Verzeichnis

       Maske = A*.E24                   -> alle Fots die mit 'A' beginnen.
