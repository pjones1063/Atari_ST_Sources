SCAN4PIC allows you to examine program (and data) files for hidden pictures, 
and then save them out as a DEGAS compatable picture file.  Simply load 
SCAN4PIC.TOS and enter the name of the file that you want to examine.  Press 
the HELP key for a list of controls.  Press Control-C to exit during 
filename input to exit.
